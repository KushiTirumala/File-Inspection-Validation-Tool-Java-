File Inspection & Validation Tool (Java)

A Java console application that inspects files and validates them based on existence, readability, writability, and expected file extension. Useful for developers and data engineers to quickly verify file integrity before processing.

Project Overview

This project demonstrates Java I/O operations, file handling, and validation logic. Users can input a file path, get its properties, and validate it against expected criteria like extension type.

Features

Inspect file properties: name, size, path, readability, writability

Validate file existence and type

Check if file matches the expected extension

Console-based interactive interface

Handles invalid or missing files gracefully

Technologies Used

Java SE

File I/O (java.io.File)

Console input/output (Scanner)

Object-oriented programming
